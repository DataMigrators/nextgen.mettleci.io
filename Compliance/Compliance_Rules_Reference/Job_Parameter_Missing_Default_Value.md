# Job Parameter Missing Default Value

|                  |                                                |
|------------------|------------------------------------------------|
| **Rule name**    | Job Parameter Missing Default Value            |
| **Parallel Job** | Yes                                            |
| **Server Job**   | Yes                                            |
| **Job Sequence** | Yes                                            |
| **Description**  | Identify Job parameters within a default value |

# Description

It is good practice to consider the use of default parameter values to
protect jobs from unintended behaviour when invoked without providing
all required parameter values.

# Actions

Ensure all your Job parameters have a default value.
