# Job Parameter Not in a Parameter Set

|                  |                                                                     |
|------------------|---------------------------------------------------------------------|
| **Rule name**    | Job Parameter Not in a Parameter Set                                |
| **Parallel Job** | Yes                                                                 |
| **Server Job**   | Yes                                                                 |
| **Job Sequence** | Yes                                                                 |
| **Description**  | Identifies Job parameters which are not members of a parameter sets |

# Description

Identifies Job parameters which are not members of a parameter sets.
Environment variables are permitted to exist outside parameter sets.

# Actions

Ensure Job parameters are members of a parameter sets.
