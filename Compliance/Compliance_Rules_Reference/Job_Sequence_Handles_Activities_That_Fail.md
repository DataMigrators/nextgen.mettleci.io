# Job Sequence Handles Activities That Fail

|                  |                                                                      |
|------------------|----------------------------------------------------------------------|
| **Rule name**    | Job Sequence Handles Activities That Fail                            |
| **Parallel Job** | \-                                                                   |
| **Server Job**   | \-                                                                   |
| **Job Sequence** | Yes                                                                  |
| **Description**  | Verifies that a job sequence automatically handles failed activities |

# Description

Sequence jobs can be configured to allow DataStage to automatically
handle jobs that fail within a sequence. You can enable or disable
automatic failure handling recording at a project-wide level, or for
individual sequence jobs. This compliance rule verifies that this
setting has been set for a specified job sequence.

# Actions

Ensure that automatic failure handling has been set for the specified
Job Sequence.
