# Schema Files

|                  |                                |
|------------------|--------------------------------|
| **Rule name**    | Schema Files                   |
| **Parallel Job** | Yes                            |
| **Server Job**   | \-                             |
| **Job Sequence** | \-                             |
| **Description**  | Detect the use of Schema Files |

# Description

In the Upgrade scenario, Schema Files are file system assets that need
to be managed and migrated separately. This rule identifies references
to schema files which you will need to manage.

# Actions

Ensure your schema files are governed in the same manner as your other
DataStage assets.
