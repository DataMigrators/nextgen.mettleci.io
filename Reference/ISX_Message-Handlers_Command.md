# ISX Message-Handlers Command

# Purpose

Inject Job-level Message Handlers into ISX files that then will be
imported into DataStage.

# Example

``` java
$> mettleci isx message-handlers \
   -isx isx_dir \
   -msh msh_handler_dir
```
