# Purpose

A command line mechanism for generating a Report Card for organisations
who cannot allow their ISX's to be exported from their internal
network. 

# Example

``` bash
$> mettleci compliance reportcard \
   -zipfile assets/SC/ProdGL-3projects.zip \
   -reportdir reports \
   -client “Freddie Mercury” \
   -email “frederick.mercury@datamigrators.com”
```
