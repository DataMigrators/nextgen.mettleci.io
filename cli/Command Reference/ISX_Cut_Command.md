# ISX Cut Command

Cuts an ISX File containing the definition of multiple DataStage assets
into multiple ISX files, each containing the definition of a single
DataStage asset.

# Example

``` bash
$> mettleci isx cut \ 
   -dir output_dir \
   -isx cut_test.isx 
```
