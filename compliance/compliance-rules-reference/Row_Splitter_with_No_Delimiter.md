# Row Splitter with No Delimiter

|                  |                                                                                       |
|------------------|---------------------------------------------------------------------------------------|
| **Rule name**    | Row Splitter with No Delimiter                                                        |
| **Parallel Job** | No                                                                                    |
| **Server Job**   | Yes                                                                                   |
| **Job Sequence** | \-                                                                                    |
| **Description**  | Identifies Row Splitter Stage with output format delimited but no delimiter specified |

# Description

A Row Splitter Stage has an output format set as **delimited** but no
delimiter is specified.

# Actions

Ensure that Row Splitter Stages with an output format set as
**delimited** have a delimiter is specified.
