# Manual Build User Recipient

The **MettleCI - Notification Extensions** installs a Manual Build User
Recipient type which facilitates notifying the user who manually
triggered a build with regards to it's status

## Configure a Manual Build User Recipient

Steps

1.  Navigate to the **Notifications** configuration tab for the job
    (this will be the default job if creating a new plan).

2.  Click the pencil icon of an existing Notification, or click **Add
    Notification**.

3.  If you can't find the **Manual Build User Recipient** type, check
    the add-on **MettleCI - Notification Extensions**
    (dm-bamboo-notification.jar) has been installed and is enabled.

4.  Complete the **Add Build Notification** settings form with the
    appropriate **Event** and **Recipient Type** values.

5.  Click **Save**
