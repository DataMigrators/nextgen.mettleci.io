# Data Fabrication Functions - Random

A collection of random data fabrication functions that don’t fit in to
other categories. You can use some of these to specify
organisation-specific values you want to use as a source of fabricated
columns.

|            |                |                                          |                |
|------------|----------------|------------------------------------------|----------------|
| **Module** | **Fabricator** | **Description**                          | **Parameters** |
| random     | alphaNumeric   | A random alphanumeric character          |                |
|            | arrayElement   |                                          |                |
|            | boolean        |                                          |                |
|            | image          |                                          |                |
|            | locale         |                                          |                |
|            | number         | A random number within a specified range |                |
|            | objectElement  |                                          |                |
|            | uuid           |                                          |                |
|            | word           | A random word                            |                |
|            | words          | Some random words                        |                |
